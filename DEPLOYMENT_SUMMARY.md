## 🎯 Next Steps

### For Users

1. Clone the repository:

   ```bash
   git clone https://github.com/jackstealer/Jatayuu.git
   cd Jatayuu
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   cd dashboard && npm install
   ```

3. Run the system:

   ```bash
   # Terminal 1: Backend
   python main.py

   # Terminal 2: Dashboard
   cd dashboard && npm run dev
   ```
